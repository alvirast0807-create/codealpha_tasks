# Sales Prediction using Python

## Objective
Predict product sales based on advertising expenditures across different media channels.

## Dataset
Advertising Dataset containing:

- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget
- Sales

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm

Linear Regression

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Correlation Analysis
4. Model Training
5. Sales Prediction
6. Model Evaluation

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## Key Insights

- TV advertising has the highest impact on sales.
- Radio advertising contributes positively to sales.
- Newspaper advertising has comparatively lower influence.
- Businesses can optimize marketing budgets using sales prediction.
