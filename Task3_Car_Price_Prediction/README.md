# Car Price Prediction with Machine Learning

## Objective
Predict car selling prices using machine learning techniques.

## Dataset
Car Price Dataset containing:
- Car Name
- Year
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner Information

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm
Random Forest Regressor

## Project Workflow
1. Data Cleaning
2. Feature Engineering
3. Data Visualization
4. Model Training
5. Model Evaluation
6. Price Prediction

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
